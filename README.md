# HackerTyper terminal
_A clone of the Hacker-Typer from http://hackertyper.net but for the terminal_

## Manuel usage

### windows
```
iwr "https://raw.githubusercontent.com/louishermanpaelinck/hackertyper/refs/heads/main/hackertyper.sh" | Select -Expand Content
```

### linux
```
curl -s https://raw.githubusercontent.com/louishermanpaelinck/hackertyper/refs/heads/main/hackertyper.sh
```
